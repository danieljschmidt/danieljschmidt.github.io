S0 = 10^5; % burn-in replications
S = 10^6; % number of draws from the Gibbs sampler that are not discarded)
% define vectors for the draws from the Gibbs sampler
gibbs_theta1 = zeros(S0+S,1);
gibbs_theta2 = zeros(S0+S,1);
gibbs_T0 = zeros(S0+S,1);
% vector of possible break points
T0_vec = (1:T-1)';
% initialization
T0 = ceil(rand(1)*T-1); % uniform distribution over 1, ... T-1
for s=1:S0+S
% sample theta1 and theta2 given T0
n1 = sum(y(1:T0));
n2 = sum(y(T0+1:end));
theta1 = betarnd(n1+alpha, T0-n1+delta);
theta2 = betarnd(n2+alpha, T-T0-n2+delta);
% sample T0 given theta1 and theta2
n1_vec = cumsum(y(1:T-1));
n2_vec = sum(y) - n1_vec;
logw1 = log(theta1)*(n1_vec)+log(1-theta1)*(T0_vec-n1_vec);
logw2 = log(theta2)*(n2_vec)+log(1-theta2)*(-T0_vec-n2_vec);
logweights = logw1+logw2;
logweights = logweights-max(logweights); % rescale logweights such that max weights = 1
weights = exp(logweights); % convert rescaled logweights to weights
cumprobs = cumsum(weights)/sum(weights); % vector that represents cumulative distribution function
T0 = sum(rand(1)>cumprobs)+1;
% save draws
gibbs_theta1(s) = theta1;
gibbs_theta2(s) = theta2;
gibbs_T0(s) = T0;
end
Since this is only a toy example with very low dimensionality, I do not conduct a rigorous check of convergence. A quick look at the trace plots suggests that convergence to the posterior occurs very fast and the dependence on the initialization of the break point becomes negligible for small values of s. Thus the chosen number of burn-in replications seems to be overly conservative.

Final Remarks

It is important to keep in mind that all results above are conditional on the basic assumption that there is just one structural break in the Bernoulli process. The multimodal structure of the posterior distribution of break points might indicate that this assumption is too restrictive. A fully Bayesian model in which the number of structural breaks is no longer fixed may reveal a more nuanced picture on how the precision of forecasts has changed over time.
